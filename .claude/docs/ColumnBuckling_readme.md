# 기둥 좌굴 허용 사용하중 계산기

AISC(미국철강구조학회) 설계 기준에 따라 강재 기둥의 **최대 허용 사용하중(ton)** 을 계산하는 .NET 8 콘솔 애플리케이션입니다.  
핀-핀(양단 힌지) 경계 조건 기준으로 동심 하중 및 편심 하중 두 가지 케이스를 모두 지원합니다.

---

## 요구 사항

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

---

## 빌드 및 실행

```bash
# 빌드
dotnet build

# 인자 없이 실행 (기본값: 300A PIPE, 길이 4470mm, 편심 25%)
dotnet run

# 인자 지정: [부재명] [기둥길이(mm)] [편심비율(소수)]
dotnet run -- "300A PIPE" 4470 0.25
dotnet run -- "H300x300" 6000 0
dotnet run -- "150 I.A (150x90x9/12)" 3000 0.10

# 배포용 단일 실행파일 빌드
dotnet publish -c Release
```

> **편심비율 입력 단위**: 소수로 입력합니다 (예: `0.25` = 25%).  
> **편심 = 0** 이면 순수 압축 공식, **편심 > 0** 이면 Secant Formula가 적용됩니다.

## 계산 원리

### 기본 설계 파라미터 (기본값)

| 파라미터 | 기호 | 기본값 |
|---|---|---|
| 탄성계수 | E | 210,000 MPa |
| 항복응력 | Fy | 240 MPa |
| 안전율 | SF | 3.0 |
| 경계 조건 | K | 1.0 (핀-핀) |

### Step 1. 편심량(e) 계산

I.A 단면과 그 외 단면의 편심 기준거리 산정 방식이 다릅니다.

```
일반 단면 : e = 편심비율 × (B / 2)
I.A 단면  : e = 편심비율 × (ReferenceDim − c_y)
```

> I.A 단면은 비대칭이므로 도심(`c_y`)을 기준으로 편심거리를 보정합니다.  
> I.A 단면은 부재명에 `.`이 포함된 경우로 자동 식별합니다.

### Step 2. 탄성 좌굴임계응력 (Euler 공식)

$$F_e = \frac{\pi^2 E I}{L^2 A}$$

### Step 3. 비탄성 좌굴임계응력 (AISC 기준)

세장비 한계값을 계산하여 두 가지 식 중 하나를 적용합니다.

$$\lambda_{limit} = 4.71\sqrt{\frac{E}{F_y}}$$

| 조건 | 적용 식 |
|---|---|
| $KL/r \leq \lambda_{limit}$ (비탄성 좌굴) | $F_{cr} = 0.658^{F_y/F_e} \cdot F_y$ |
| $KL/r > \lambda_{limit}$ (탄성 좌굴) | $F_{cr} = 0.877 \cdot F_e$ |

### Step 4. 허용 사용하중 계산

**편심 = 0 (동심 하중):**

$$P_{allow} = \frac{F_{cr} \cdot A}{SF \times 9810}$$

**편심 > 0 (편심 하중, Secant Formula):**

하중 비율을 100%에서 0.1%까지 0.1% 단위로 감소시키며, 아래 조건을 최초로 만족하는 `P`를 최대 허용하중으로 결정합니다.

$$\sigma_{max} = \frac{P}{A} \left(1 + \frac{A \cdot e}{Z} \cdot \sec\left(\frac{L}{2r}\sqrt{\frac{P}{AE}}\right)\right) \leq F_y$$

$$P_{allow} = \frac{P_{max}}{SF \times 9810}$$

> 9,810 N/ton 환산 상수 적용. 전체 범위에서 조건을 만족하지 못하면 예외를 발생시킵니다.


### 단면 데이터 파일 (`PropertyRefer.txt`)

탭(`\t`) 구분 텍스트 파일로, 아래 열 순서로 데이터를 관리합니다.

| 인덱스 | 내용 |
|---|---|
| 0 | 부재명 (Key) |
| 2 | B 또는 내경 d (mm) |
| 3 | 단면적 A (mm²) |
| 5 | 약축 관성모멘트 Iy (mm⁴) |
| 7 | 약축 회전반경 ry (mm) |
| 9 | 약축 단면계수 Zy (mm³) |
| 11 | 도심 c_y (mm) — I.A 단면만 해당 |

새 단면을 추가할 경우 이 파일에 동일한 열 순서로 행을 추가하면 됩니다.
